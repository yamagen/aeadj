# 前置き型発動表現

Last change: 2026/06/12-15:32:04.

## 要旨

本稿は、AEAD No.11「おどろいたことに、...」を出発点として、発話の冒頭に置かれ、後続内容の受け取り方を先に指定する表現を「前置き型発動表現」として整理するためのメモである。

「おどろいたことに」「残念なことに」のような感情・評価を先に置く表現、「一般的に」「常識的に」「個人的に」のように適用範囲や立場を先に限定する表現、「それに加えて」「最後に」のように談話構造を先に示す表現は、いずれも後続発話の受け取り方を予告する。これらは活用を必要とせず、文頭に置きやすく、非母語話者にとっても利用しやすい。

本稿では、これらの表現を、単なる副詞句や談話標識としてではなく、発話を立ち上げながら、次に述べる内容を組み立てる時間を話し手に与える発動単位として位置づける。

## キーワード

AEAD, 即時文法, 前置き型発動表現, 感情前置き, 限定前置き, 構造前置き, 談話標識, スタンス標識, 定型表現

## 1. 出発点

AEAD No.11 の中心表現は次である。

```text
おどろいたことに、...
```

この表現は、後続する出来事や事実を述べる前に、話し手の反応を先に提示する。

```text
おどろいたことに、彼は日本人なみに日本語を話す。
```

ここで「おどろいたことに」は、後続内容そのものの一部ではなく、後続内容をどのように受け取るべきかを先に指定する。聞き手は、次に来る内容を「驚くべきこと」として受け取る準備をする。

この型は、次のように広がる。

```text
おもしろいことに、...
うれしいことに、...
残念なことに、...
```

これらは、後続内容に対する感情・評価を先に置く表現である。

## 2. 前置き型発動表現

本稿では、次のような表現を仮に「前置き型発動表現」と呼ぶ。

```text
前置き型発動表現:
  後続発話の感情・評価・範囲・立場・構造を先に示し、
  発話を開始しながら、話し手に次の内容を組み立てる時間を与える定型的表現。
```

重要なのは、これらが単に意味を足すだけではなく、発話を始めるための足場になることである。非母語話者にとって、言いたい内容をすべて組み立ててから話し始めるのは難しい。前置き型表現は、発話を開始しつつ、後続内容を準備するための時間を与える。

## 3. 分類案

### 3.1 感情・評価前置き型

後続内容に対する話し手の感情や評価を先に出す型である。

```text
おどろいたことに、...
おもしろいことに、...
うれしいことに、...
残念なことに、...

surprisingly, ...
interestingly, ...
fortunately, ...
unfortunately, ...
```

この型では、聞き手は後続内容を、単なる事実としてではなく、話し手が驚いたこと、面白いと見たこと、残念だと見たこととして受け取る。

### 3.2 限定・立場前置き型

後続内容の適用範囲、一般性、立場、主観性を先に限定する型である。

```text
一般的に、...
常識的に、...
個人的に、...
私の考えでは、...
正直に言うと、...
はっきり言うと、...

generally speaking, ...
strictly speaking, ...
personally, ...
in my opinion, ...
to be honest, ...
frankly speaking, ...
```

この型は、後続内容をどの範囲で受け取ればよいかを先に指定する。たとえば「個人的に、...」と言えば、後続内容は一般論ではなく、話し手個人の見解として提示される。

### 3.3 構造前置き型

談話の構造を先に示す型である。

```text
それに加えて、...
一方で、...
そのため、...
最後に、...
まとめると、...

in addition to that, ...
on the other hand, ...
for that reason, ...
to conclude, ...
to sum up, ...
```

この型は、後続発話が追加なのか、対比なのか、理由なのか、結論なのかを予告する。聞き手は、次の発話の役割を先に予測できる。

## 4. 活用しないことの利点

これらの表現の多くは、活用を必要としない。たとえば「一般的に」「個人的に」「最後に」は、後続内容にかかわらず、そのまま文頭に置くことができる。

これは非母語話者にとって大きな利点である。動詞活用、助詞選択、複文構成を考える前に、まず発話を開始できるからである。

```text
一般的に、...
個人的に、...
残念なことに、...
それに加えて、...
最後に、...
```

これらは、多少場違いに使われても、発話そのものを破綻させにくい。もちろん、過剰使用や不自然な使用はありうるが、発話開始の支援装置としては非常に利用しやすい。

## 5. 時間稼ぎとしての働き

前置き型発動表現は、聞き手に後続内容の受け取り方を予告するだけでなく、話し手自身に時間を与える。

```text
一般的に、...
そうですね、個人的に言うと、...
残念なことに、...
それに加えて、...
```

話し手は、これらを発している間に、次に述べる内容を組み立てることができる。これは単なるフィラーではない。発話はすでに始まっており、しかも後続内容の方向づけも行われている。

したがって、これらは「ええと」「あの」のような純粋なためらい表現とは異なる。時間を稼ぎながら、同時に談話上の機能を果たしている。

## 6. 既存研究との接続

この観察は、いくつかの既存研究と接続できる。

第一に、談話標識の研究である。Schiffrin (1987) は、談話標識が会話の局所的・全体的なまとまりの解釈を助けるものとして論じている。Fraser (1999) も、談話標識を、隣接する発話や文の関係を示す表現として扱っている。

第二に、文副詞・スタンス副詞・スタンス標識の研究である。Biber et al. (1999) は副詞的要素を、状況を補足するもの、話し手の立場や評価を示すもの、接続関係を示すものに分けている。Hyland (2005) は、学術的談話において書き手がどのようにスタンスを示し、読者との関係を作るかを論じている。

第三に、定型表現・formulaic language の研究である。Wray (2002) や Conklin and Schmitt (2012) は、定型表現が言語処理や流暢性と深く関係することを論じている。前置き型発動表現は、単語単位ではなく、発話を始めるための定型的な発動単位として扱うことができる。

ただし、本稿の関心は、既存研究の分類をそのまま採用することではない。AEAD の観点では、これらを「発話開始」「時間稼ぎ」「後続内容の受け取り方の予告」という時間的・行動的な側面から捉え直す。

## 7. AEAD における位置づけ

AEAD では、前置き型発動表現を、語彙知識ではなく、発話を立ち上げる発動単位として扱う。

たとえば、学習者が次のように話し始める場合、その学習者は単に「選択肢」という語を知っているだけではない。

```text
人生にはいくつか選択肢があって、...
```

この発話は、複数の可能性を整理して語り始めるための型を持っていることを示す。これは、日本語が上手に見える重要な要因である。

同様に、次のような前置き表現も、発話の立ち上げに利用できる。

```text
一般的に、...
個人的に、...
おもしろいことに、...
それに加えて、...
最後に、...
```

これらは、活用を必要とせず、文頭に置きやすく、後続内容を考える時間を生み出す。AEAD では、このような表現を、非母語話者が自然に発話を開始するための重要な資源として記録する価値がある。

## 9. 既存研究との差異

既存研究では、これらの表現は談話標識、スタンス標識、文副詞、定型表現などとして扱われてきた。これらの分類は、表現が後続発話にどのような意味的・談話的関係を与えるかを説明する上で有効である。

しかし、本稿の関心は、分類そのものではなく、話し手がそれらをどのように利用して発話を開始するかにある。前置き型発動表現は、後続内容の感情・評価・範囲・構造を予告するだけでなく、話し手が次に述べる内容を組み立てる時間を作る。

この点で、前置き型発動表現は、単なる談話標識ではなく、発話生成のための足場である。特に非母語話者にとっては、動詞活用や複雑な文構成を始める前に、まず発話を立ち上げることができる点が重要である。
